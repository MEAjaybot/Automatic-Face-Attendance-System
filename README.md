# Automatic-Face-Attendance-System
Plus attendance analysis
<br>
<p>A real-time face recognition-based attendance system built using OpenCV and MTCNN + InceptionResnetV1(FaceNet).</p>
<br>
<p>The system automatically detects faces from camera feed, generates embeddings, matches them with stored encodings, and marks attendance.</p>
<br>
<h>Features</h>
1.Real time face detection using MTCNN
2.Face recognition using InceptionResnetV1(PyTorch)
3.Automatic aatendance logging(CSV per day)
4.Easy student enrollment through image upload
5.Attendance reports with date filters
6.pie charts and punctuality analysis
7.interactive UI built in Streamlit
<br>

<h>Technologies Used
1.Face Detection(MTCNN)
2.Face embedding(InceptionResnetV1)faceNet
3.UI(streamlit)
4.Video Processing OpenCV
5.Data Storage(CSV)

# Automatic-Face-Attendance-System
Plus attendance analysis
<br>
<p>A real-time face recognition-based attendance system built using OpenCV and MTCNN + InceptionResnetV1(FaceNet).</p>
<br>
<p>The system automatically detects faces from camera feed, generates embeddings, matches them with stored encodings, and marks attendance.</p>
<br>
##Features
-Real time face detection using MTCNN
-Face recognition using InceptionResnetV1(PyTorch)
-Automatic aatendance logging(CSV per day)
-Easy student enrollment through image upload
-Attendance reports with date filters
-pie charts and punctuality analysis
-interactive UI built in Streamlit
<br>

##Technologies Used
-Face Detection(MTCNN)
-Face embedding(InceptionResnetV1)faceNet
-UI(streamlit)
-Video Processing OpenCV
-Data Storage(CSV)
